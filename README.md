# CUPA-Installer
GrapheneOS Cross User Profile App Installer

# Overview
The idea is to use a 2nd profile with FAGoogle services to install apps, but don't run them in that 2nd profile, instead, copy/install them in the first profile without any FAGoogle services. The copy step reqiured a USB-Flash drive of some sort to be plugged into the smart phone. 

At least this is the idea for now!

# Why?
Avoid installing FAGoogle services in the main profile, but get the apps (apks) that do run without FAGoogle services.
